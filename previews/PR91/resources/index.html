Resources

Targeted Learning is a difficult topic, while it is not strictly necessary to understand the details to use this package it can certainly help. Here is an incomplete list of external resources that I found useful in my journey.

Websites

These are two very clear introductions to causal inference and semi-parametric estimation:

  • Introduction to Modern Causal Inference (Alejandro Schuler, Mark J. van der Laan).
  • A Ride in Targeted Learning Territory (David Benkeser, Antoine Chambaz).

Text Books

  • Targeted Learning (Mark J. van der Laan, Sherri Rose).

Journal articles

  • Semiparametric doubly robust targeted double machine learning: a review (Edward H. Kennedy).
